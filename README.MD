# OCEANFT PRICE CALCULATE

This project is using for calculate 7 Days average assets price

## How to install

`$ npm install`

## How to run and build

`$ npm run dev`
`$ npm run build`

## How to deploy

- Create static web site eg. S3 or Firebase
- Select and move all file from `out directory`
  - out directory
    - _next/*
    - 404.html
    - index.html
    - Other
